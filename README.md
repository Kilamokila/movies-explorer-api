# movies-explorer-api

*Node.js v.16.14.0
*Express.js v.4.17.2
*MongoDB
*Mongoose v.6.2.1
*HTML
*CSS
*JavaScript
*React.js
*CRA
*JSX
*React DOM
*React Router
*БЭМ Nested
*Git
*Webpack
